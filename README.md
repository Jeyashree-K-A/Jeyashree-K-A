🛒 **Shoppy**  
✨Shoppy is a full-stack e-commerce web application developed using the MERN stack, allowing users to browse products, add items to cart, and place orders.  
✨The project integrates frontend and backend functionalities using RESTful APIs and MongoDB for data management.  
✨Product images are uploaded and managed using Cloudinary, enabling efficient cloud-based image storage and retrieval.  
✨Email notifications are implemented using Brevo (Sendinblue) to send order confirmations and updates, enhancing user communication and experience.

⚡ **PulsePoint**  
✨PulsePoint is a full-stack MERN web application developed with a focus on efficient frontend rendering and smooth backend API integration.  
✨The frontend is built using React with Vite to ensure fast loading, optimized performance, and a clean component-based structure.  
✨The backend is implemented using Node.js and Express.js, handling server-side logic, API requests, and data flow between the client and database.  
✨MongoDB is used for managing application data, enabling structured storage and retrieval of information.  
✨This project was developed to strengthen end-to-end full-stack development skills, including frontend–backend communication and application architecture.
